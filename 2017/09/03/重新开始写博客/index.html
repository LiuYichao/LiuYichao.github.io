开始写博客

写博客的目的

  1. 记录自己在工作中遇到的困难和解决办法，因为很多时候即使自己做过一遍，经常也会忘记，所以索性记下来，给自己回顾
  2. 希望做一个展示自己的地方，把自己的学习方法等展示出来，希望能帮助别人，也希望别人看到后觉得我不足的地方给我建议
  3. 我这人做事有个毛病，没有长性，以前也写过博客，但是当时写的东西都太简单，就放弃了，现在我知道即使再简单的东西也有记下来的必要，

博客的内容

  1. 自然语言的学习（包括一些英文文章翻译，和日语的学习）
  2. 编程相关的内容
  3. 可能会有一些生活的随笔
